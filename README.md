# 모임 소개
서울 을지로 인근 (광화문, 종로, 서울역, 서울시청, 남산 등) 에서 이뤄지고 있는 주제에 제한이 없는 모임입니다. 반드시 을지로 인근 직장에 다니실 필요는 없으며 학생/직장인 구분 없이 참여 가능합니다.
AWS 초보, 숙련자 모두 모여서 공유하고 싶은 이야기, 논의 하고 싶은 이야기 나누며 자유롭게 소통해요
